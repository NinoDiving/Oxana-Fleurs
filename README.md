# Oxana Fleurs 🌸

Bienvenue sur **Oxana Fleurs**, une boutique en ligne dédiée à la vente de fleurs fraîches et de compositions florales élégantes.

## 🌿 Description

Oxana Fleurs est une plateforme e-commerce qui permet aux utilisateurs de découvrir et d'acheter des fleurs pour toutes les occasions. L'interface intuitive et moderne permet une navigation fluide et une expérience d'achat agréable.

## 🚀 Fonctionnalités

- 🛍️ **Catalogue de fleurs** : Parcourez une sélection variée de fleurs et bouquets.
- 🔍 **Recherche avancée** : Trouvez rapidement vos fleurs préférées grâce à un système de filtres.
- 📦 **Gestion des commandes** : Ajoutez des articles à votre panier et passez commande en toute simplicité.
- 🖼️ **Système de login** : Permet au utilisateur de s'inscrire pour facilité les achats.
- 🖼️ **Images des produits** : Chaque bouquet est présenté avec une photo détaillée.
- 🎨 **Back-office admin** : Interface d'administration pour gérer les produits, les commandes.

## 🛠️ Technologies utilisées

- **Frontend** : React (TSX)
- **Backend** : Node.js avec Express
- **Base de données** : MySQL


## 📌 Installation et lancement

### 1️⃣ Cloner le projet
```sh
git clone https://github.com/votre-utilisateur/oxana-fleurs.git
cd oxana-fleurs
```

### 2️⃣ Installer les dépendances
```sh
cd client && npm install
cd ../server && npm install
```

### 3️⃣ Lancer le monorepo
```sh
cd racine-projet
npm run dev
```


Le frontend sera accessible sur `http://localhost:3000` et le backend sur `http://localhost:3310`.
